This website is built using my new knowledge in HTML and CSS with a sprinkle of PHP and JavaScript. The purpose of this site is to create a similar feel to my portfolio. To create my portfolio, I used Squarespace to drag and drop the features and layout of the site. As I am applying to UI/UX positiong, I realize how important having an updated portfolio is. I purchased my domain and then got to work on highlighting some of the graphic designs and webpagesI have created in my current role as a web design intern.
I have had some time to search the web for UX design tools which lead me to find Lottie Animations. My company does not use this resource currently so I have been using the free animations to embed into Duda's HTML feature. I think this adds another layer of content that can catch a viewer. I have became a regular reader of Wired and UX Planet to find more tools for designs. I have found Land Book which is a super cool website inspiration database.
